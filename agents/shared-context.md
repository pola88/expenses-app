# Shared Context - Expense App

# Product Overview

Personal finance and expense tracking application.

Goal:
Help users understand and manage their finances.

---

# Core Features (MVP)

- Add/edit/delete expenses
- Categories
- Monthly summaries
- Expense filtering
- Exchange money (USD Mep to Pesos Argentinos)

---

# Future Features

- Budgets
- Notifications
- Reports
- Recurring expenses

---

# Domain Considerations

- Money precision
- Date-based aggregations
- Currency handling
- Summary consistency

---
